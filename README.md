# lucky_svg_sprite_cli.cr

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     lucky_svg_sprite_cli:
       github: tilitribe/lucky_svg_sprite_cli.cr
   ```

2. Run `shards install`

## Usage

```crystal
require "lucky_svg_sprite_cli"
```
